Retail Sales Analysis & Dashboard
📊 Overview

This project analyzes retail sales data to identify revenue trends, product performance, and overall business insights. The analysis was performed using SQL, Python, and Power BI to provide a complete end-to-end data analytics solution.

🛠️ Tools & Technologies
SQL (Data Analysis & Querying)
Python (Pandas, NumPy, Matplotlib for EDA)
Power BI (Dashboard & Visualization)
Excel (Visualization)

📈 Key Insights
Total revenue reached ~323K, with moderate profitability
Sales show a sharp decline after initial months, followed by stable performance
Revenue is concentrated in a few product categories, indicating dependency
Sales distribution is even across cities, with no dominant region
One branch slightly outperforms others in overall revenue
Customer ratings (~7 average) indicate scope for service improvement
📉 Data Analysis (Python)
Performed Exploratory Data Analysis (EDA) using Pandas and Matplotlib
Analyzed monthly revenue trends and category-wise performance
Visualized patterns in sales, quantity, and customer ratings
🧠 SQL Analysis Highlights
Calculated revenue, profit, and sales performance metrics
Analyzed product-wise and category-wise sales contribution
Identified monthly trends and performance variations
